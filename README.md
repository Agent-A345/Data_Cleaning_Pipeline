## 🏠 Airbnb Data Cleaning Pipeline

A complete data cleaning and preprocessing project built using Python and Jupyter Notebook, focused on transforming raw Airbnb listing data into a clean, structured, and analysis-ready dataset through handling missing values, duplicates, inconsistent formatting, and feature standardization.

## 🚀 Features
* **Comprehensive Data Cleaning Pipeline** – End-to-end preprocessing workflow for Airbnb datasets.
* **Missing Value Handling** – Detects and treats null or incomplete data entries.
* **Duplicate Removal System** – Identifies and removes duplicate records efficiently.
* **Data Type Standardization** – Converts inconsistent columns into proper formats.
* **Outlier Detection & Treatment** – Handles abnormal values for improved data quality.
* **String & Text Cleaning** – Cleans categorical and textual data fields.
* **Feature Engineering Support** – Prepares data for future analytics and machine learning tasks.
* **Data Consistency Validation** – Ensures uniformity across dataset columns.
* **Exploratory Checks & Summaries** – Generates insights into cleaned dataset structure.
* **Scalable Preprocessing Workflow** – Reusable cleaning steps for large datasets.

## 🛠️ How to Run
1. **Clone the repository**:
```
git clone https://github.com/Agent-A345/Data_Cleaning_Pipeline.git
```
2. **Navigate to the project folder**:
```
cd Data_Cleaning_Pipeline
```
3. **Install required libraries**
```
pip install pandas numpy matplotlib seaborn jupyter
```
4. **Run the notebook**
```
jupyter notebook
```
5. **Open**
```
Airbnb_Data_Cleaning_Pipeline.ipynb
```
